Pointer events

En kort gennemgang

Pointer events er forskellige måder man kan behandle input på, i form af mus, touch og pen.

De forskellige pointer events er som følgende:

  • pointerdown
  • pointerup
  • pointermove
  • pointerover
  • pointerout
  • pointerenter
  • pointerleave
  • pointercancel
  • gotpointercapture
  • lostpointercapture

De forskellige events reagerer på hver deres input. F.eks. så reagerer pointerdown når musen bliver trykket, og når man trykker med sin finger på touch-devices
